# AlphaRadar

AlphaRadar is a research project designed to detect early stage crypto opportunities.

The project tracks signals from developer activity, emerging ecosystems, and social growth.

The goal is to identify promising Web3 protocols before they become widely known.

---

## Signals tracked

• developer activity  
• ecosystem growth  
• early stage protocols  
• narrative trends  

---

## Vision

Build an open research engine for discovering crypto alpha.

---

## Repository Structure

scanner → protocol discovery  
research → crypto analysis  
datasets → protocol tracking  
tools → research utilities
